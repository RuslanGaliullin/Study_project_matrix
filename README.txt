Мтод для ввода с файла мб недоконца описан: вводится название, если его нет - throw
Exception, читается первая строка с размерами, если неправильно что-то - throw Exception,
аналогично при считывании матрицы по строкам, если дошли до конца файла - матрицы готова!

Мб в условиях непонятно написал, можно написать rnd вместо числа по запросу ввода
количества строк или столбцов или вместо числа при вводе элементов матрицы.
